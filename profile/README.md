## 🌌 Về Dự Án
**AuRen** là một hệ sinh thái mã nguồn mở được phát triển bởi **SandboxCodex**. Chúng tôi tập trung vào việc tạo ra các giải pháp tự động hóa, hệ điều hành môi trường ảo và các AI Agent thông minh.

### 🛠 Hệ Sinh Thái AuRen
*   **[AuRen OS](https://github.com/nguyenvanhoaithuong0507-hub/auren-os):** Nền tảng cốt lõi định nghĩa lại trải nghiệm người dùng.
*   **[AuRen Bot](https://github.com/nguyenvanhoaithuong0507-hub/auren-bot):** Trợ lý thông minh hỗ trợ vận hành và quản lý.
*   **[Sandbox Code](https://github.com/nguyenvanhoaithuong0507-hub/SandboxCode):** Môi trường thực thi mã an toàn và linh hoạt.
*   **[.auren Agent](https://github.com/AuRen-agent/.auren):** Cấu hình và bộ não điều khiển cho các Agent thế hệ mới.
